Plain jsonrpcserver
===================

Using jsonrpcserver's built-in ``serve`` method.

::

    $ pip install jsonrpcserver

The quickest way to serve a method::

    from jsonrpcserver import method
    method(ping=lambda:'pong')
    serve()

Using the ``@method`` decorator:
